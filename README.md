# Vim Configuration - Task 1

This repository contains my custom Vim setup, including a specialized colorscheme and a Git integration plugin.

### Plugins Used
* **Vundle.vim**: Used as the plugin manager to handle installation and updates. [Link](https://github.com/VundleVim/Vundle.vim)
* **Badwolf**: A dark, high-contrast colorscheme for better code visibility. [Link](https://github.com/sjl/badwolf)
* **Vim-Fugitive**: A Git wrapper that allows me to run Git commands directly inside Vim. [Link](https://github.com/tpope/vim-fugitive)
